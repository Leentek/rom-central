AetherSX2 - PlayStation 2 Emulator for Android

Important Status Update: Development of AetherSX2 was stopped by the original developer. The app has been removed from the Google Play Store and will not receive further official updates. Users wanting to use AetherSX2 must find older, trusted APK versions (e.g., build 3668 'alpha-2230') from reliable archive sources online. **Exercise caution and avoid downloading from untrusted websites, as unofficial APKs may contain malware.**
BIOS Required: AetherSX2 requires PlayStation 2 BIOS files (e.g., SCPH-39001.bin, SCPH-70012.bin) to function. You must legally obtain these files from your own PS2 console.

Android Setup Guide

AetherSX2 brings PS2 emulation to Android devices, based on the PCSX2 project. It requires powerful hardware for good performance.

  1. Find & Download Trusted APK - Since it's not on the Play Store, search reputable emulation communities or archive sites for recommended older AetherSX2 builds (like build 3668). Download the APK file.
  2. Enable Unknown Sources & Install - Go to your Android Settings > Security (or Apps > Special app access) > "Install unknown apps" and grant permission to your browser or file manager. Then, open the downloaded APK file and tap "Install".
  3. Obtain & Place BIOS - Legally dump the BIOS files from your PS2 console. Launch AetherSX2 once. It will likely ask for the BIOS location during setup, or you can set it later in App Settings > BIOS. A common location is to create a `bios` folder inside `Internal Storage/Android/data/xyz.aethersx2.android/files/` and place the `.bin` files there.
  4. Launch & Initial Setup - Open AetherSX2. Grant storage permissions. Follow the setup wizard: select theme, point to your BIOS directory, and add your game directory (where your PS2 ISO, CHD, or CSO files are stored).
  5. Configure Settings - Access 'App Settings' from the side menu:
    • `Graphics`: Select Renderer (`Vulkan` preferred if supported, otherwise `OpenGL`), set Upscale Multiplier (start at `1x` or `2x`), Aspect Ratio, etc.
    • `System`: Adjust `EE Cycle Rate` (underclocking, e.g., -1 or -2) and `EE Cycle Skip` (underclocking) can improve speed in some demanding games but may cause issues. Experiment cautiously.
    • `Controls`: Configure the `Touchscreen` controller layout, or go to `Controller Port 1` to map a connected `Gamepad`.
  6. Load Game - Your games should appear in the main list after setting the game directory. Tap a game to start playing.

System Requirements (Android): Very Demanding. A high-end device with a powerful **ARM64 processor (Snapdragon 845 or newer strongly recommended)**, at least 4GB RAM (more is better), and support for Vulkan graphics is needed for acceptable performance in many PS2 games.

Frequently Asked Questions

Why was AetherSX2 discontinued and removed from the Play Store?

The original developer stopped development citing various reasons including online harassment and issues with ad implementation impersonation. It was subsequently removed from the Google Play Store.

What is the best version/build of AetherSX2 to use?

Since development stopped, many users recommend sticking with specific well-regarded older builds obtained from trusted archive sources, such as build 3668 (often labelled alpha-2230), which predates some controversial changes. Always verify the source of your APK.

Is the PS2 BIOS file really mandatory?

Yes. AetherSX2 requires official PlayStation 2 BIOS files dumped legally from your own console to boot games. There is no HLE BIOS alternative included.

How can I improve performance in AetherSX2?

Ensure you have a powerful enough device. Use the Vulkan renderer if supported. Start with native resolution (1x upscale) and increase cautiously. Experiment with lowering the EE Cycle Rate and EE Cycle Skip in System settings for demanding games (e.g., -1/-1 or -2/-1), but note this can cause instability or glitches. Close background apps.

Is there a PS2 emulator for PC?

Yes, AetherSX2 is based on the long-standing and excellent open-source PC emulator **PCSX2**. If you want to play PS2 games on Windows, Linux, or macOS, PCSX2 is the recommended choice.

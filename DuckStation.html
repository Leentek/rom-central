DuckStation - Accurate PlayStation (PSX/PS1) Emulator

Important: DuckStation requires PlayStation 1 BIOS files to function. You must legally obtain these from your own console (e.g., `scph5501.bin` for USA). Place them in the 'bios' folder within your DuckStation directory or where prompted during setup.
Note: DuckStation focuses on accuracy and modern features. It has built-in graphics/sound/input handling, so **no external plugins are needed** like with older emulators (e.g., ePSXe). It supports features like resolution upscaling, PGXP geometry correction (fixes polygon wobble), texture filtering, and more.

PC Setup Guide (Windows, Linux, macOS)

DuckStation offers a modern and feature-rich PS1 emulation experience on desktop platforms.

  1. Download DuckStation - Get the latest stable or preview build for your OS from the official DuckStation GitHub releases page.
  2. Extract Files - Extract the downloaded archive (`.zip`, `.tar.xz`) into a dedicated folder.
  3. Obtain & Place BIOS - Acquire your PS1 BIOS files legally. Create a `bios` subfolder inside your DuckStation folder and place the BIOS files (e.g., `scph5500.bin`, `scph5501.bin`, `scph5502.bin`) inside it.
  4. Run & Initial Setup - Launch the DuckStation executable. A first-time setup wizard should appear:
    • Select Language & Theme.
    • Confirm or browse to your `bios` folder path.
    • Click `Add Game Directory` and select the folder(s) containing your PS1 game images (BIN/CUE, ISO, IMG, PBP, CHD, MDF/MDS).
  5. Configure Settings (Optional) - Access `Settings` from the menu bar:
    • `Display Settings`: Choose Renderer (Vulkan, D3D11/12, OpenGL, Metal), Aspect Ratio, VSync.
    • `Enhancement Settings`: Set Internal Resolution Scale (e.g., 3x for 1080p), Texture Filtering, Widescreen Hack, and enable PGXP options (Geometry Correction, Culling Correction, Texture Correction) for improved visuals.
    • `Controller Settings`: Select controller type (e.g., DualShock), choose your device, and map buttons automatically or manually.
  6. Load Game - Your scanned games will appear in the main window list. Double-click a game to start playing.

System Requirements (PC): Relatively lightweight for native resolution. A CPU from the last decade and integrated graphics should suffice. However, using enhancements like resolution scaling (3x+) and PGXP requires a more capable, modern graphics card (supporting Vulkan/D3D11+/OpenGL 3.1+).

Android Setup Guide

DuckStation provides an excellent, feature-rich PS1 experience on Android devices.

  1. Install DuckStation - Install the 'DuckStation' app from the Google Play Store or download the APK from the official GitHub releases page (enable "Install unknown apps" if needed).
  2. Grant Permissions & Place BIOS - Launch the app and grant storage permissions. Place your legally obtained PS1 BIOS file(s) in a location on your device storage (e.g., a dedicated `DuckStation/bios` folder you create).
  3. Initial Setup - Follow the on-screen setup wizard:
    • The app will ask you to specify the location of your BIOS files. Browse to and select the folder where you placed them.
    • Add your game directory/directories containing your PS1 game images.
  4. Configure Settings (Optional) - Access the app settings menu (often via a side drawer or gear icon):
    • `Graphics Settings`: Select Renderer (Vulkan recommended if supported), Upscale Multiplier, Aspect Ratio.
    • `Enhancement Settings`: Enable PGXP options, Texture Filtering, Widescreen Hack if desired and if your device can handle it.
    • `Control Settings`: Customize the touchscreen overlay layout and opacity, or map buttons for a connected Bluetooth/USB gamepad.
  5. Load Game - Your scanned games will appear in the main app list. Tap a game to start playing.

System Requirements (Android): Runs well on a wide range of devices for native resolution. Mid-range devices can often handle some upscaling and basic enhancements. High-end devices (Snapdragon 8xx series or equivalent) are recommended for higher upscaling factors and full PGXP features.

Frequently Asked Questions

Is the PS1 BIOS file mandatory for DuckStation?

Yes. DuckStation focuses on accuracy and requires official Sony PlayStation BIOS files (dumped legally from your own console) to boot games.

Why doesn't DuckStation need plugins like ePSXe?

DuckStation is a more modern emulator with its graphics (Vulkan, D3D, OpenGL, Metal, Software), sound, and input systems built directly into the application. This eliminates the need for users to find and configure separate external plugin files.

What is PGXP and should I enable it?

PGXP (Parallel/Precision Geometry Transform Pipeline) is a feature unique to emulators like DuckStation that corrects graphical issues inherent to the original PS1 hardware, such as wobbling polygons and distorted textures. Enabling PGXP options (Geometry Correction, Texture Correction, Culling Correction) significantly improves visual quality but requires more GPU power.

Is DuckStation better than ePSXe or Beetle PSX?

DuckStation is widely regarded as one of the best PS1 emulators available today, offering a great balance of accuracy, performance, features (like PGXP), and ease of use. Beetle PSX (Mednafen core) is known for extremely high accuracy but can be more demanding. ePSXe is older and relies on plugins but still has good compatibility. For most users, DuckStation is often the recommended choice now.

What game formats does DuckStation support?

DuckStation supports most common PS1 formats, including BIN/CUE (single or multiple BINs), ISO, IMG, MDF/MDS, PBP, ECM, and CHD (recommended compressed format).

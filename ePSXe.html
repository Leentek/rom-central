ePSXe - PlayStation (PSX/PS1) Emulator Setup Guide

Important Requirements:
1. ePSXe requires a PlayStation 1 BIOS file (e.g., scph1001.bin) obtained legally from your own console.
2. The PC version requires separate GPU, SPU, and CDR plugin files to be downloaded and configured.

PC Setup Guide (Windows / Linux)

ePSXe for PC relies on external plugins for rendering graphics, sound, and handling CD drives.

  1. Download ePSXe - Get the latest version for your OS from the official ePSXe website (epsxe.com).
  2. Download Plugins - You need GPU, SPU, and CDR plugins. Search online for reputable sources like Pete Bernert's website (historical) or emulation plugin archives. Common choices include:
    • **GPU:** Pete's OpenGL2 Driver, Pete's D3D Driver
    • **SPU:** P.E.Op.S. SPU, Eternal SPU Plugin
    • **CDR:** ePSXe CDR WNT/W2K core, Pete's CDR Driver
  3. Extract & Organize - Extract the ePSXe archive into a folder. Inside that folder, create two new folders: `bios` and `plugins`. Extract/place the downloaded plugin files (`.dll` on Windows) into the `plugins` folder.
  4. Obtain & Place BIOS - Acquire your PS1 BIOS file legally (e.g., `scph1001.bin`). Place it inside the `bios` folder you created.
  5. Run & Configure ePSXe - Launch `ePSXe.exe`. Use the `Config` > `Wizard Guide` for easy first-time setup, or configure manually:
    • `Config` > `Bios`: Click `Select...` and choose your BIOS file from the `bios` folder.
    • `Config` > `Video`: Select your chosen GPU plugin (e.g., Pete's OpenGL2). Click `Configure` to set resolution, texture filtering, etc.
    • `Config` > `Sound`: Select your SPU plugin and click `Configure`.
    • `Config` > `Cdrom`: Select your CDR plugin and configure if necessary.
    • `Config` > `Game Pads` > `Port 1` > `Pad 1`: Select controller type (e.g., DualShock) and map buttons for your keyboard or gamepad.
  6. Load Game - Use `File` > `Run ISO` to load a game image from your hard drive (common formats: ISO, BIN/CUE, IMG, PBP, ECM, CHD). Or use `File` > `Run CDROM` if playing from a physical disc (requires CDR plugin setup).

System Requirements (PC): Generally low for basic PS1 emulation. A dual-core CPU and any graphics card supporting the chosen GPU plugin (OpenGL 2.0+ or DirectX 9+) should work. Higher resolutions or advanced plugin features may require more power.

Android Setup Guide

ePSXe for Android is a paid application available on the Google Play Store. It simplifies plugin handling compared to the PC version but still requires a BIOS file.

  1. Purchase & Install ePSXe - Buy and install 'ePSXe for Android' from the Google Play Store.
  2. Obtain & Place BIOS - Acquire your PS1 BIOS file legally. Create a folder named `epsxe` in your device's internal storage, and inside that, create a folder named `bios`. Place the BIOS file (e.g., `scph1001.bin`) into this `Internal Storage/epsxe/bios/` folder using a file manager app.
  3. Launch & Configure - Open the ePSXe app. It should automatically detect the BIOS if placed correctly. If not, go to `Preferences` > `Bios File` and manually select it.
  4. Adjust Preferences - Explore `Preferences`:
    • `CPU Preferences`: Adjust CPU frameskip if needed (usually not).
    • `Video Preferences`: Choose screen orientation, aspect ratio, video renderer (Hardware or Software).
    • `Input Preferences`: Configure touch screen controls (Player 1 -> `Touchscreen gamepad`), map a physical gamepad, or adjust settings.
  5. Load Game - Tap `Run Game`. ePSXe will scan your device storage for compatible game images (ISO, BIN, PBP, etc.). Tap the game you want to play from the list.

System Requirements (Android): Relatively lightweight. Most modern Android devices (Android 4.0+) should run ePSXe well. Performance depends more on the specific game and chosen settings.

Frequently Asked Questions

Is the PS1 BIOS file really mandatory for ePSXe?

Yes. Unlike some emulators with High-Level Emulation (HLE) options, ePSXe strictly requires an official Sony PlayStation BIOS file (dumped legally from your own console) to function.

Where do I get plugins for ePSXe on PC?

ePSXe doesn't bundle plugins. You need to download them separately. Historically, Pete Bernert's website was a primary source. You can search online forums and emulation sites for reputable plugin downloads (e.g., Pete's OpenGL2/D3D, P.E.Op.S SPU, various CDR plugins).

Is the Android version of ePSXe free?

No, ePSXe for Android is a paid application on the Google Play Store. There is no official free version.

How does ePSXe compare to DuckStation or Beetle PSX?

ePSXe is an older, established emulator known for its compatibility and performance, relying on a plugin system for PC. DuckStation and Beetle PSX (Mednafen core often used in RetroArch) are more modern, often open-source alternatives focusing on high accuracy, built-in features (no external plugins needed), and advanced options like PGXP geometry correction. Many users now prefer DuckStation for its ease of use and accuracy.

What game formats does ePSXe support?

ePSXe typically supports ISO, BIN/CUE, IMG, PBP (PSP eboot format for PS1 games), ECM (compressed format requiring unecm tool), and sometimes CHD (compressed format).
